# Metrics

This index is a progressive-disclosure entry point for this part of the GTFS Schedule OKF bundle.

| concept | type | description |
|---|---|---|
| [service span](service-span.md) | Metric | The first-to-last scheduled departure window for a service date, route, direction, or stop pattern. |
| [trips per route service day](trips-per-route-service-day.md) | Metric | Count of scheduled trips by route and active service date. |
| [average headway](average-headway.md) | Metric | Average departure spacing for a route, direction, stop, or frequency window. |
| [peak headway](peak-headway.md) | Metric | The representative headway during a defined peak period. |
| [stop count](stop-count.md) | Metric | Count of rider boarding/alighting stops in a feed or route subset. |
| [route count](route-count.md) | Metric | Count of distinct rider-facing routes in routes.txt. |
| [calendar service date coverage](calendar-service-date-coverage.md) | Metric | The set and span of service dates represented by calendar.txt and calendar_dates.txt. |
