# Systems

This index is a progressive-disclosure entry point for feed catalogs, validators, routing engines, and community-resource systems that help agents use GTFS in practice.

| concept | type | description |
|---|---|---|
| [mobility database](mobility-database.md) | System | MobilityData's global catalog for discovering GTFS, GTFS Realtime, and GBFS feeds. |
| [transitland](transitland.md) | System | Catalog, archive, and API platform for discovering and inspecting public transit feeds. |
| [canonical gtfs schedule validator](canonical-gtfs-schedule-validator.md) | System | MobilityData's validator for checking GTFS Schedule feeds against the official reference and best practices. |
| [open trip planner](open-trip-planner.md) | System | Open-source multimodal trip planner that can build routing graphs from GTFS and OpenStreetMap inputs. |
| [gtfs community resources](gtfs-community-resources.md) | System | GTFS.org resource hub for community tools, datasets, standards, apps, and research around public transit open technology. |
